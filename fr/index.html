Projets pertinents

Pannier d'achat

Petite application web de commerce électronique, d'une seule page construite avec React. Ce site web comprend une page d'acceuil, et une page d'achat avec un panier d'achat interactif à barre latérale. Il conserve une conception adaptive visible sur tous les appareils

Application météo

Application web JavaScript construite à l'aide d'une API météo, qui affiche la météo du jour, les prévisions météorologiques pour les prochaines 21 heures, etc. Ça comprend des arrière-plans dynamiques qui changent en fonction de la météo actuelle, et une barre de recherche, qui permet à l'utilisateur l'accès à la météo mondiale

Bataille navale

Application web JavaScript créée à l'aide du développement piloté par les tests avec Jest. Un joueur humain affronte un simple joueur IA dans une partie de bataille navale. L'IA possède la capacité de discerner ses propres coups réussis, et de tenter de frapper les segments restants du navire.

Liste de tâches

Application web JavaScript qui permet à un utilisateur de planifier à l'avance. L'utilisateur peut ajouter des catégories de tâches, et ajouter des tâches incluant le nom de la tâche, la catégorie, la date d'échéance, la priorité et la description. Cela comprend une Interface élégante avec une barre latérale basculante.
